# All-India Electronics Retail Performance Tracker

## Project Overview
An interactive Excel dashboard designed to analyze retail sales performance across multiple states in India. The project tracks Key Performance Indicators (KPIs) like Total Sales, Average Order Value, and Average Discount alongside brand performance metrics.

## Key Features
* **Dynamic KPI Cards:** Live calculation of core sales metrics.
* **Geographical Distribution:** An interactive map highlighting state-wise revenue.
* **Brand & Category Analysis:** Visual breakdown of sales trends by brand and product category.

## Technologies Used
* **Microsoft Excel:** Pivot Tables, Advanced Formulas, Dynamic Shapes, and Slicers.
